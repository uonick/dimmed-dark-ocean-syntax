# Dimmed collection

[Sublime Text 2/3](https://git.io/dimmed)

[Atom](https://git.io/v78CS)


## Dimmed Dark Ocean theme based on Ocean Syntax
There is also a matching [UI theme](https://atom.io/themes/one-dark-ui).

### Support syntax
* PHP
* PYTHON
* JAVASCRIPT
* GO
* JAVA
* RUBY
* INI
* CSS
* JSON
* HTML
* VOLT
