## Dimmed Dark Ocean theme based on One Dark Syntax
There is also a matching [UI theme](https://atom.io/themes/one-dark-ui).

### Support syntax
* PHP
* PYTHON
* JAVASCRIPT
* GO
* JAVA
* RUBY
* INI
* CSS
* JSON
* HTML
* VOLT
